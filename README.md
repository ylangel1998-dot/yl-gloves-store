# The GloveStory - Luxury Gloves E-commerce

Artisan luxury gloves crafted with passion. Heritage meets contemporary design.

## Tech Stack

- **Frontend**: React 18 + Cloudflare Pages
- **Backend**: Supabase (PostgreSQL + Auth)
- **Deployment**: Cloudflare Workers + Pages
- **Payments**: Pagsmile / Airwallex (Coming Soon)

## Getting Started

```bash
# Install dependencies
npm install

# Development
npm run dev

# Deploy
npm run deploy
```
